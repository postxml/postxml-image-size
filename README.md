# postxml-image-size
[![npm version][npm-image]][npm-url]

> [PostXML] plugin adds sizes to images.

## Installation
`npm i postxml-image-size --save-dev`

## Usage
[Postxml]

## Options
### round
**Enable rounding sizes**
Type: `String`<br>
Default: `true`

## Example

### Input
```html
<img src="image.png">
```

### Output
```html
<img src="image.png" width="255" height="180">
```
